UON Parsers

The {@link oaj.uon.UonParser} class is used to parse UON into POJOs.

The UON parser provides the following settings:

  • {@doc juneau-marshall.ConfigurableProperties.Common Common Properties}
  • {@doc juneau-marshall.ConfigurableProperties.CommonParser Common Parser Properties}
  • {@link oaj.uon.UonParser}
    • {@link oaj.uon.UonParser#UON_decoding UON_decoding}
    • {@link oaj.uon.UonParser#UON_validateEnd UON_validateEnd}

The following pre-configured parsers are provided for convenience:

  • {@link oaj.uon.UonParser}
    • {@link oaj.uon.UonParser#DEFAULT DEFAULT}
    • {@link oaj.uon.UonParser#DEFAULT_DECODING DEFAULT_DECODING}
