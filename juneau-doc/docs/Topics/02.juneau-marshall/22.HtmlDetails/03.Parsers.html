HTML Parsers

The {@link oaj.html.HtmlParser} class is used to parse HTML into POJOs. They can also parse the contents produced by {@link oaj.html.HtmlDocSerializer}.

The HTML parser provides the following settings:

  • {@doc juneau-marshall.ConfigurableProperties.Common Common Properties}
  • {@doc juneau-marshall.ConfigurableProperties.CommonParser Common Parser}

The following pre-configured parsers are provided for convenience:

  • {@link oaj.html.HtmlParser}
    • {@link oaj.html.HtmlParser#DEFAULT DEFAULT}
