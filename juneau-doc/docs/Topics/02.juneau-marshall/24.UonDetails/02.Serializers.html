UON Serializers

The {@link oaj.uon.UonSerializer} class is used to serialize POJOs into UON.

The UON serializers provides the following settings:

  • {@doc juneau-marshall.ConfigurableProperties.Common Common Properties}
  • {@doc juneau-marshall.ConfigurableProperties.CommonSerializer Common Serializer Properties}
  • {@link oaj.uon.UonSerializer}
    • {@link oaj.uon.UonSerializer#UON_addBeanTypes UON_addBeanTypes}
    • {@link oaj.uon.UonSerializer#UON_encoding UON_encoding}
    • {@link oaj.uon.UonSerializer#UON_paramFormat UON_paramFormat}

The following pre-configured serializers are provided for convenience:

  • {@link oaj.uon.UonSerializer}
    • {@link oaj.uon.UonSerializer#DEFAULT DEFAULT}
    • {@link oaj.uon.UonSerializer#DEFAULT_ENCODING DEFAULT_ENCODING}
    • {@link oaj.uon.UonSerializer#DEFAULT_READABLE DEFAULT_READABLE}
