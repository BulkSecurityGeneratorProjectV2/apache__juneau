XML Parsers

The {@link oaj.xml.XmlParser} class is used to parse XML into POJOs.

The XML parser provides the following settings:

  • {@doc juneau-marshall.ConfigurableProperties.Common Common Properties}
  • {@doc juneau-marshall.ConfigurableProperties.CommonParser Common Parser Properties}
  • {@link oaj.xml.XmlParser}
    • {@link oaj.xml.XmlParser#XML_eventAllocator XML_eventAllocator}
    • {@link oaj.xml.XmlParser#XML_preserveRootElement XML_preserveRootElement}
    • {@link oaj.xml.XmlParser#XML_reporter XML_reporter}
    • {@link oaj.xml.XmlParser#XML_resolver XML_resolver}
    • {@link oaj.xml.XmlParser#XML_validating XML_validating}

The following pre-configured parsers are provided for convenience:

  • {@link oaj.xml.XmlParser}
    • {@link oaj.xml.XmlParser#DEFAULT DEFAULT}
