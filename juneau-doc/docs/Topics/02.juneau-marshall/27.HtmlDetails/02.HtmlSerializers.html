{title:'HTML Serializers', flags:'todo'}

The {@link oaj.html.HtmlSerializer} class is used to serialize POJOs into HTML.

The {@link oaj.html.HtmlDocSerializer} class is the same, but wraps the serialized POJO inside a document template consisting of header, nav, aside, and footer sections.

The HTML serializers provides the following settings:

  • {@doc GlossaryConfigurableProperties Configurable Properties}
  • {@link oaj.html.HtmlSerializer.Builder}
    • {@link oaj.html.HtmlSerializer.Builder#addBeanTypes() addBeanTypes()}
    • {@link oaj.html.HtmlSerializer.Builder#addKeyValueTableHeaders() addKeyValueTableHeaders()}
    • {@link oaj.html.HtmlSerializer.Builder#disableDetectLabelParameters() disableDetectLabelParameters()}
    • {@link oaj.html.HtmlSerializer.Builder#disableDetectLinksInStrings() disableDetectLinksInStrings()}
    • {@link oaj.html.HtmlSerializer.Builder#labelParameter(String) labelParameter(String)}
    • {@link oaj.html.HtmlSerializer.Builder#uriAnchorText(AnchorText) uriAnchorText(AnchorText)}
  • {@link oaj.html.HtmlDocSerializer.Builder}
    • {@link oaj.html.HtmlDocSerializer.Builder#aside(String[]) #aside(String[])}
    • {@link oaj.html.HtmlDocSerializer.Builder#asideFloat(AsideFloat) #asideFloat(AsideFloat)}
    • {@link oaj.html.HtmlDocSerializer.Builder#footer(String[]) footer(String[])}
    • {@link oaj.html.HtmlDocSerializer.Builder#head(String[]) head(String[])}
    • {@link oaj.html.HtmlDocSerializer.Builder#header(String[]) header(String[])}
    • {@link oaj.html.HtmlDocSerializer.Builder#nav(String[]) nav(String[])}
    • {@link oaj.html.HtmlDocSerializer.Builder#navlinks(String[]) navlinks(String[])}
    • {@link oaj.html.HtmlDocSerializer.Builder#noResultsMessage(String) noResultsMessage(String)}
    • {@link oaj.html.HtmlDocSerializer.Builder#nowrap() nowrap()}
    • {@link oaj.html.HtmlDocSerializer.Builder#script(String[]) script(String[])}
    • {@link oaj.html.HtmlDocSerializer.Builder#style(String[]) style(String[])}
    • {@link oaj.html.HtmlDocSerializer.Builder#stylesheet(String[]) stylesheet(String[])}
    • {@link oaj.html.HtmlDocSerializer.Builder#template(Class) template(Class)}
    • {@link oaj.html.HtmlDocSerializer.Builder#widgets(Class...) widgets(Class...)}

The following pre-configured serializers are provided for convenience:

  • {@link oaj.html.HtmlSerializer}
    • {@link oaj.html.HtmlSerializer#DEFAULT }
    • {@link oaj.html.HtmlSerializer#DEFAULT_SQ }
    • {@link oaj.html.HtmlSerializer#DEFAULT_SQ_READABLE }
  • {@link oaj.html.HtmlDocSerializer}
    • {@link oaj.html.HtmlDocSerializer#DEFAULT }
