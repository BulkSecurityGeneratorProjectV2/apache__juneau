{title:'HTML Serializers', flags:'todo'}

The {@link oaj.html.HtmlSerializer} class is used to serialize POJOs into HTML.

The {@link oaj.html.HtmlDocSerializer} class is the same, but wraps the serialized POJO inside a document template consisting of header, nav, aside, and footer sections.

The HTML serializers provides the following settings:

  • {@doc GlossaryConfigurableProperties Configurable Properties}
  • {@link oaj.html.HtmlSerializerBuilder}
    • {@link oaj.html.HtmlSerializerBuilder#addBeanTypes() addBeanTypes()}
    • {@link oaj.html.HtmlSerializerBuilder#addKeyValueTableHeaders() addKeyValueTableHeaders()}
    • {@link oaj.html.HtmlSerializerBuilder#disableDetectLabelParameters() disableDetectLabelParameters()}
    • {@link oaj.html.HtmlSerializerBuilder#disableDetectLinksInStrings() disableDetectLinksInStrings()}
    • {@link oaj.html.HtmlSerializerBuilder#labelParameter(String) labelParameter(String)}
    • {@link oaj.html.HtmlSerializerBuilder#uriAnchorText(AnchorText) uriAnchorText(AnchorText)}
  • {@link oaj.html.HtmlDocSerializerBuilder}
    • {@link oaj.html.HtmlDocSerializerBuilder#aside(String[]) #aside(String[])}
    • {@link oaj.html.HtmlDocSerializerBuilder#asideFloat(Float) #asideFloat(Float)}
    • {@link oaj.html.HtmlDocSerializerBuilder#footer(String[]) footer(String[])}
    • {@link oaj.html.HtmlDocSerializerBuilder#head(String[]) head(String[])}
    • {@link oaj.html.HtmlDocSerializerBuilder#header(String[]) header(String[])}
    • {@link oaj.html.HtmlDocSerializerBuilder#nav(String[]) nav(String[])}
    • {@link oaj.html.HtmlDocSerializerBuilder#navlinks(String[]) navlinks(String[])}
    • {@link oaj.html.HtmlDocSerializerBuilder#noResultsMessage(String) noResultsMessage(String)}
    • {@link oaj.html.HtmlDocSerializerBuilder#nowrap() nowrap()}
    • {@link oaj.html.HtmlDocSerializerBuilder#script(String[]) script(String[])}
    • {@link oaj.html.HtmlDocSerializerBuilder#style(String[]) style(String[])}
    • {@link oaj.html.HtmlDocSerializerBuilder#stylesheet(String[]) stylesheet(String[])}
    • {@link oaj.html.HtmlDocSerializerBuilder#template(Class) template(Class)}
    • {@link oaj.html.HtmlDocSerializerBuilder#widgets(Class...) widgets(Class...)}

The following pre-configured serializers are provided for convenience:

  • {@link oaj.html.HtmlSerializer}
    • {@link oaj.html.HtmlSerializer#DEFAULT }
    • {@link oaj.html.HtmlSerializer#DEFAULT_SQ }
    • {@link oaj.html.HtmlSerializer#DEFAULT_SQ_READABLE }
  • {@link oaj.html.HtmlDocSerializer}
    • {@link oaj.html.HtmlDocSerializer#DEFAULT }
