{title:'BasicRestObject', created:'8.1.0', updated:'9.0.0'}

The {@link oajr.BasicRestObject} class is identical to the {@link oajr.BasicRestServlet} class except that it does not extend from HttpServlet. It defines the exact same set of serializers, parsers, etc., but it cannot be deployed as a top-level servlet. It can however be used for child resources registered via the {@link oajr.annotation.Rest#children() @Rest(children)} annotation explained later.

This class is particularly important in Spring Boot environments where you want to defined REST objects as Spring Beans. Spring Boot will automatically deploy any classes that extend from Servlet as top-level resources. Extending from this class will prevent that behavior.

The code for this class is virtually identical to {@link oajr.BasicRestServlet} but lacks a parent class:

BasicRestServlet.java

@Rest public abstract class BasicRestObject implements BasicUniversalRest, BasicRestOperations { @Override /* BasicRestOperations */ public Swagger getSwagger(RestRequest req) { return req.getSwagger().orElseThrow(NotFound::new); } @Override /* BasicRestOperations */ public HttpResource getHtdoc(@Path("/*") String path, Locale locale) throws NotFound { return getContext().getStaticFiles().resolve(path, locale).orElseThrow(NotFound::new); } @Override /* BasicRestOperations */ public HttpResource getFavIcon() { String favIcon = getContext().getConfig().getString("REST/favicon", "images/juneau.png"); return getHtdoc(favIcon, null); } @Override /* BasicRestOperations */ public void error() {} @Override /* BasicRestOperations */ public RestContextStats getStats(RestRequest req) { return req.getContext().getStats(); } }
