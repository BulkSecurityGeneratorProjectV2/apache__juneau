{title:'@Response / @ResponseHeader / @ResponseStatus', updated:'8.1.0,9.0.0'}

The {@link oaj.http.annotation.Response @Response} / {@link oaj.http.annotation.ResponseHeader @ResponseHeader} / {@link oaj.http.annotation.ResponseStatus @ResponseStatus} annotations allow you to define method parameters for setting HTTP response parts:

  • {@link oaj.http.annotation.Response}
    • {@link oaj.http.annotation.Response#() api()}
    • {@link oaj.http.annotation.Response#() code()}
    • {@link oaj.http.annotation.Response#() description()}
    • {@link oaj.http.annotation.Response#() example()}
    • {@link oaj.http.annotation.Response#() examples()}
    • {@link oaj.http.annotation.Response#() headers()}
    • {@link oaj.http.annotation.Response#() on()}
    • {@link oaj.http.annotation.Response#() onClass()}
    • {@link oaj.http.annotation.Response#() parser()}
    • {@link oaj.http.annotation.Response#() schema()}
    • {@link oaj.http.annotation.Response#() serializer()}
    • {@link oaj.http.annotation.Response#() value()}
  • {@link oaj.http.annotation.ResponseHeader}
    • {@link oaj.http.annotation.ResponseHeader#code() code()}
    • {@link oaj.http.annotation.ResponseHeader#description() description()}
    • {@link oaj.http.annotation.ResponseHeader#name() name()}
    • {@link oaj.http.annotation.ResponseHeader#on() on()}
    • {@link oaj.http.annotation.ResponseHeader#onClass() onClass()}
    • {@link oaj.http.annotation.ResponseHeader#schema() schema()}
    • {@link oaj.http.annotation.ResponseHeader#serializer() serializer()}
    • {@link oaj.http.annotation.ResponseHeader#value() value()}
  • {@link oaj.http.annotation.ResponseStatus}
    • {@link oaj.http.annotation.ResponseStatus#on() on()}
    • {@link oaj.http.annotation.ResponseStatus#onClass() onClass()}

Response part parameters are defined using a settable {@link oaj.Value} object.

The {@link oaj.http.annotation.Response @Response} annotation can be used to set the response body as a settable parameter:

@RestGet("/testResponseBody") public void testResponseBody( @Response( serializer=OpenApiSerialier.class ) @Schema( items=@Items( collectionFormat="pipes", items=@SubItems( collectionFormat="csv", type="integer", format="int64", minimum="0", maximum="100" minLength=1, maxLength=10 ) ), minLength=1, maxLength=10 ) Value<Long[][]> responseBody ) {...}

The {@link oaj.http.annotation.ResponseHeader} annotation can be used to set response headers:

@RestGet("/testResponseHeader") public void testResponseHeader( @ResponseHeader("My-Header") @Schema( collectionFormat="pipes", items=@SubItems( collectionFormat="csv", type="integer", format="int64", minimum="0", maximum="100" minLength=1, maxLength=10 ), minLength=1, maxLength=10 ) Value<Long[][]> header ) { header.set(new Long[][]{...}); }

The {@link oaj.http.annotation.ResponseStatus @ResponseStatus} annotation annotation can be used to set the HTTP response code:

Examples:

// Defined on parameter. @RestGet("/user/login") public void login( @FormData("username") String username, @FormData("password") String password, @ResponseStatus Value<Integer> status ) { if (! isValid(username, password)) status.set(401); }
