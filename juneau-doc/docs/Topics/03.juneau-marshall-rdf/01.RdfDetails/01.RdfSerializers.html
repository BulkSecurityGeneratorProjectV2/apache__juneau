    • {@link oaj.jena.N3Serializer}
    • {@link oaj.jena.NTripleSerializer}

The following pre-configured serializers are provided for convenience:

  • {@link oaj.jena.RdfSerializer}
    • {@link oaj.jena.RdfXmlSerializer}
      • {@link oaj.jena.RdfXmlSerializer#DEFAULT DEFAULT}
    • {@link oaj.jena.RdfXmlAbbrevSerializer}
      • {@link oaj.jena.RdfXmlAbbrevSerializer#DEFAULT DEFAULT}
    • {@link oaj.jena.N3Serializer}
      • {@link oaj.jena.N3Serializer#DEFAULT DEFAULT}
    • {@link oaj.jena.NTripleSerializer}
      • {@link oaj.jena.NTripleSerializer#DEFAULT DEFAULT}
    • {@link oaj.jena.TurtleSerializer}
      • {@link oaj.jena.TurtleSerializer#DEFAULT DEFAULT}

Abbreviated RDF/XML is currently the most widely accepted and readable RDF syntax, so the examples shown here will use that format.
