{title:'RDF Parsers', updated:'9.0.0'}

The {@link oaj.jena.RdfParser} class is the top-level class for all Jena-based parsers. Language-specific parsers are defined as inner subclasses of the RdfParser class:

  • {@link oaj.jena.RdfParser}
    • {@link oaj.jena.RdfXmlParser}
    • {@link oaj.jena.NTripleParser}
    • {@link oaj.jena.TurtleParser}
    • {@link oaj.jena.N3Parser}

The RdfParser.Xml parser handles both regular and abbreviated RDF/XML.

The class hierarchy for the builder of this parser is:

  • {@link oaj.Context.Builder}
    • {@link oaj.BeanContextable.Builder}
      • {@link oaj.parser.Parser.Builder}
        • {@link oaj.parser.ReaderParser.Builder}
          • {@link oaj.jena.RdfParser.Builder}

Refer to the builder javadocs for configurable settings.

The following pre-configured parsers are provided for convenience:

  • {@link oaj.jena.RdfParser}
    • {@link oaj.jena.RdfXmlParser}
      • {@link oaj.jena.RdfXmlParser#DEFAULT DEFAULT}
    • {@link oaj.jena.N3Parser}
      • {@link oaj.jena.N3Parser#DEFAULT DEFAULT}
    • {@link oaj.jena.NTripleParser}
      • {@link oaj.jena.NTripleParser#DEFAULT DEFAULT}
    • {@link oaj.jena.TurtleParser}
      • {@link oaj.jena.TurtleParser#DEFAULT DEFAULT}
