Response headers are accessed through the following methods:

  • {@link oajrc.RestResponse}
    • {@link oajrc.RestResponse#getHeader(String) getHeader(String)} returns {@link oajrc.RestResponseHeader}
    • {@link oajrc.RestResponse#getHeaders(String) getHeaders(String)} returns {@link oajrc.RestResponseHeader}[]
    • {@link oajrc.RestResponse#getFirstHeader(String) getFirstHeader(String)} returns {@link oajrc.RestResponseHeader}
    • {@link oajrc.RestResponse#getLastHeader(String) getLastHeader(String)} returns {@link oajrc.RestResponseHeader}
    • {@link oajrc.RestResponse#getAllHeaders() getAllHeaders()} returns {@del oaj.client2.RestResponseHeader}[]
    • {@link oajrc.RestResponse#getStringHeader(String) getStringHeader(String)} returns String
    • {@link oajrc.RestResponse#containsHeader(String) containsHeader(String)} returns boolean

Unlike {@link oajrc.RestResponse#getFirstHeader(String)} and {@link oajrc.RestResponse#getLastHeader(String)}, the {@link oajrc.RestResponse#getHeader(String)} method returns an empty {@link oajrc.RestResponseHeader} object instead of returning null. This allows it to be used more easily in fluent calls.

Example:

| // See if response contains Location header. | boolean hasLocationHeader = client.get(URI).complete().getHeader("Location").exists();

The {@link oajrc.RestResponseHeader} class extends from the HttpClient {@link org.apache.http.Header} class and provides several convenience methods:

  • {@link oajrc.RestResponseHeader}
    • {@link oajrc.RestResponseHeader#exists() exists()} returns boolean
    • {@link oajrc.RestResponseHeader#asString() asString()} returns String
    • {@link oajrc.RestResponseHeader#asOptionalString() asOptionalString()} returns Optional<String>
    • {@link oajrc.RestResponseHeader#asStringOrElse(String) asStringOrElse(String)} returns String
    • {@link oajrc.RestResponseHeader#as(Type,Type...) as(Type,Type...)} returns T
    • {@link oajrc.RestResponseHeader#as(Class) as(Class<T>)} returns T
    • {@link oajrc.RestResponseHeader#asOptional(Type,Type...) asOptional(Type,Type...)} returns Optional<T>
    • {@link oajrc.RestResponseHeader#asOptional(Class) asOptional(Class<T>)} returns Optional<T>
    • {@link oajrc.RestResponseHeader#asMatcher(Pattern) asMatcher(Pattern)} returns {@link java.util.regex.Matcher}
    • {@link oajrc.RestResponseHeader#asMatcher(String) asMatcher(String)} returns {@link java.util.regex.Matcher}
    • {@link oajrc.RestResponseHeader#asHeader(Class) asHeader(Class<T extends BasicHeader> c)} returns {@link oaj.http.BasicHeader}
    • {@link oajrc.RestResponseHeader#asStringHeader() asStringHeader()} returns {@link oaj.http.header.BasicIntegerHeader}
    • {@link oajrc.RestResponseHeader#asIntegerHeader() asIntegerHeader()} returns {@link oaj.http.header.BasicStringHeader}
    • {@link oajrc.RestResponseHeader#asLongHeader() asLongHeader()} returns {@link oaj.http.header.BasicLongHeader}
    • {@link oajrc.RestResponseHeader#asDateHeader() asDateHeader()} returns {@link oaj.http.header.BasicDateHeader}
    • {@link oajrc.RestResponseHeader#asCsvArrayHeader() asCsvArrayHeader()} returns {@link oaj.http.header.BasicCsvArrayHeader}
    • {@link oajrc.RestResponseHeader#asEntityTagArrayHeader() asEntityTagArrayHeader()} returns {@link oaj.http.header.BasicEntityTagArrayHeader}
    • {@link oajrc.RestResponseHeader#asStringRangeArrayHeader() asStringRangeArrayHeader()} returns {@link oaj.http.header.BasicStringRangeArrayHeader}
    • {@link oajrc.RestResponseHeader#asUriHeader() asUriHeader()} returns {@link oaj.http.header.BasicUriHeader}

The {@link oajrc.RestResponseHeader#schema(HttpPartSchema)} method allows you to perform parsing of OpenAPI formats for header parts.

Example:

| // Parse the header "Foo: bar|baz". | List<String> fooHeader = client | .get(URI) | .complete() | .getHeader("Foo").schema(T_ARRAY_PIPES).as(List.class, String.class);

Assertion methods are also provided for fluent-style calls:

  • {@link oajrc.RestResponseHeader}
    • {@link oajrc.RestResponseHeader#assertString() assertString()} returns {@link oaj.assertions.FluentStringAssertion}
    • {@link oajrc.RestResponseHeader#assertInteger() assertInteger()} returns {@link oaj.assertions.FluentIntegerAssertion}
    • {@link oajrc.RestResponseHeader#assertLong() assertLong()} returns {@link oaj.assertions.FluentLongAssertion}
    • {@link oajrc.RestResponseHeader#assertDate() assertDate()} returns {@link oaj.assertions.FluentDateAssertion}

Note how in the following example, the fluent assertion returns control to the {@link oajrc.RestResponse} object after the assertion has been completed:

Example:

| // Assert the response content type is any sort of JSON. | String body = client.get(URI) | .run() | .getHeader("Content-Type").assertString().matchesSimple("application/json*") | .getContent().asString();
