8.1.2 (TBD)

Juneau 8.1.2 is ...

juneau-marshall
  • New convenience methods:
    • {@link oaj.ObjectMap.parse(String)}
    • {@link oaj.ObjectList.parse(String)}
  • {@link oaj.marshall.CharMarshall} and {@link oaj.marshall.StreamMarshall} now have public constructors.
  • {@link oaj.annotation.Beanp @Beanp} replaces {@link oaj.annotation.BeanProperty @BeanProperty} with shorter syntax.
juneau-rest-server
  • New method {@link oajr.RestResponse.setHeaderSafe(String,String)} to strip invalid characters from header values.
  • Fixed issues related to invalid characters being set on HTTP header values.
juneau-rest-client
  • {@link oajr.RestClient} is now extendible. The constructor has been made public and simplified to: {@link oajr.RestClient(RestClientBuilder)}.
  • Duplicate methods between {@link oajr.RestClientBuilder} and {@link org.apache.http.impl.client.HttpClientBuilder} have been made deprecated on the former. This eliminates the need to try to keep the two builder classes in sync.
