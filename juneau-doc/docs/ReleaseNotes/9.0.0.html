9.0.0 (TBD)

Juneau 9.0.0 is a major release.

General changes
  • Configuration property names that are enabled by default have been replaced with disabler properties.
    For example, BEAN_useInterfaceProxies which was enabled by default has been replaced with BEAN_disableUseInterfaceProxies.
juneau-marshall
  • New {@del oaj.annotation.Marshalled @Marshalled} annotation for non-bean classes.
  • New {@link oaj.annotation.Bean#example() @Bean(example)} annotation.
  • Changes to {@link oaj.annotation.Bean @Bean} annotation.
    • @Bean(bpi|bpx|bpro|bpwo) replaced with @Bean(properties|propertiesExclude|propertiesReadOnly|propertiesWriteOnly|p|xp|ro|o)
    • New {@link oaj.annotation.Bean#example() @Bean(example)} annotation.
  • Eliminated the various @XConfig(applyX={...}) annotations and replaced them with the ability to apply targeted annotations directly to configuration classes and methods (such as REST classes/methods).

    | @Rest(...) | @Bean(on="MyBean1,MyBean2",sort=true) | @UrlEncoding(onClass=MyList.class,expandedParams=true) | public class MyRestClass { | | @RestMethod(...) | @Bean(on="MyBean1,MyBean2",sort=false) | @UrlEncoding(onClass=MyList.class,expandedParams=false) | public Object myRestMethod() { ... } | }

  • New shortened form {@link oaj.http.remote.RemoteOp#value()} for specifying http method name and path.

    | // Normal form | @RemoteOp(method=PUT, path="/foo") | | // Shortened form | @RemoteOp("PUT /foo")

juneau-rest-server
  • Removed deprecated APIs.
  • Removed the @RestMethod(name) annotation. Use @RestMethod(method) instead.
  • New shortened form {@del oajr.annotation.RestMethod#value()} for specifying http method name and path.

    | // Normal form | @RestMethod(method=PUT, path="/{propertyName}") | | // Shortened form | @RestMethod("PUT /{propertyName}")

  • @RestMethod(name) annotation. Use @RestMethod(method) instead.
juneau-rest-server-springboot
juneau-rest-client
  • Replaced deprecated APIs with new RestClient API introduced in 8.2.0.
juneau-rest-mock
  • Replaced deprecated APIs with new RestClient API introduced in 8.2.0.
